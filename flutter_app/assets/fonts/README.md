# Font Assets

Download these fonts and place them in `assets/fonts/`:

## Inter Variable
- Download from: https://rsms.me/inter/
- File needed: `Inter-Variable.ttf`

## JetBrains Mono
- Download from: https://www.jetbrains.com/mono/
- Files needed:
  - `JetBrainsMono-Regular.ttf`
  - `JetBrainsMono-Bold.ttf`

## Installation

```bash
cd assets/fonts
# Download the fonts manually or use wget/curl
```

The fonts are not included in the repository due to licensing and file size.
The app will fall back to system fonts if these are not present.
